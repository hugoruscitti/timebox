# timebox
 
Este es un planificador diario, la idea es que puedas
instalarlo como una PWA en tu escritorio y usarlo para
planificar qué hacer durante el día.

Para usar la aplicación, accedé a este link y luego
seleccioná la opción "instalar como PWA" en tu navegador:

https://hugoruscitti.github.io/timebox/
